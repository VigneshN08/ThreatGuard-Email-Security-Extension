ThreatGuard is a browser-based cybersecurity project designed to detect phishing and suspicious email behavior in real time. The system integrates a Chrome extension with a monitoring dashboard to analyze email threat indicators, generate alerts, and visualize security insights.

Features
• Real-time phishing detection
• Gmail content monitoring
• Threat alert generation
• Security monitoring dashboard
• Threat analysis and reporting

Technologies
• JavaScript
• HTML
• CSS
• Chrome Extension (Manifest v3)
• Cybersecurity threat detection logic

Cybersecurity Skills
• Phishing Detection
• Threat Monitoring
• Email Security
• Threat Intelligence
• Security Analytics
